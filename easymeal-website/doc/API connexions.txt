PROCEDURE

Récupérer un token d'authentification
  faire un POST sur "/session/create.json"
  le paramètre passé doit être un hash formatté comme suit :
    { user : { password : VALUE, username : VALUE }}
  2 réponses possibles :
    - { user : "authentication needed.", message : "Username and password don't match" }
    - { message : "authentication succeeded.", auth_token : VALUE, user: VALUE }
    
Accéder au contenu
  - Mettre le token dans le header comme suit :
    Authorization : Token token=VALUE
